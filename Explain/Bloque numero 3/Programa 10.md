- Declaro la lista "colores"
``` python
colores=["rojo", "azul"]
```
- Imprimo la lista
``` python
print(colores)
```
- Utilizo el metodo `.append`
- Este metodo sirve para añadir un elemento al final de una lista
- Añado ´verde´ a la lista "colores"
``` python
colores.append("verde")
```
- Imprimo la lista "colores"
``` python
print(colores)
``` 
